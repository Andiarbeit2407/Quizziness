Login:
------------------------------------------------------------------------
Tab		->	Allgemeine Navigation
Enter		->	Login
Einfg		->	Registrieren

Hauptmenü:
------------------------------------------------------------------------
1/Q		->	Quiz starten
2/+		->	Frage hinzufügen
3/-		->	Frage löschen		
4/B		->	Bestenliste aufrufen		
Ctrl+Shift+B	->	Ausloggen
Ctrl+Shift+M	->	Musik Stummschalten
Ctrl+Shift+P	->	Musik Pausieren
Ctrl+Shift+V	->	Musik Lautstärke
Ctrl+Shift+T	->	Textfarbe ändern
Ctrl+Shift+C	->	Farbschema ändern
Ctrl+Shift+S	->	Shortcut-Fenster aufrufen


Frage hinzufügen:
------------------------------------------------------------------------
Tab		->	Allgemeine Navigation
Enter		->	Frage hinzufügen
Escape		->	Zurück zum Hauptmenü

Frage löschen:
------------------------------------------------------------------------
Tab		->	Allgemeine Navigation
Enter		->	Frage Löschen
Escape		->	Zurück zum Hauptmenü

Quiz:
------------------------------------------------------------------------
1		->	Antwort 1
2		->	Antwort 2
3		->	Antwort 3
4		->	Antwort 4
Enter/S		->	Frage Überspringen
Escape/B		->	Zurück zum Hauptmenü
